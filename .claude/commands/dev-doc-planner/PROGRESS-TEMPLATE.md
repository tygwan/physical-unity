| 아이콘 | 의미 |

| ⚠️ | 주의 필요 |

## 체크리스트 형식

```markdown
- [ ] {작업 내용} | {담당자} | {완료일 또는 -}
- [x] {작업 내용} | {담당자} | {완료일}
```

## 진행률 계산

```
완료된 체크리스트 / 전체 체크리스트 × 100
```

ASCII 프로그레스 바:
- `█` = 10% 완료
- `░` = 10% 미완료
