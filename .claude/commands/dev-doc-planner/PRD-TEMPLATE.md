## 작성 가이드라인

### 섹션별 권장 길이
- 개요: 300-500자
- 요구사항: 500-800자
- 사용자 스토리: 스토리당 200-300자
- 제약사항/의존성: 200-400자

### 우선순위 정의
- **P0**: 필수 (Must have) - 없으면 출시 불가
- **P1**: 중요 (Should have) - 가능하면 포함
- **P2**: 선택 (Nice to have) - 시간 여유시 포함

### 상태 정의
- **Draft**: 초안 작성 중
- **Review**: 리뷰 진행 중
- **Approved**: 승인됨
